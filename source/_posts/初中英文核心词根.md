中学阶段掌握核心词根词缀，能快速拓展词汇量、提升猜词能力（尤其适用于阅读完形填空）。以下整理了**中学必背的词根、前缀、后缀**，按 “含义 + 词根 / 词缀 + 高频示例词” 分类，所有示例均为中学教材核心词汇，方便直接记忆和应用：

### 一、核心词根（20 个，覆盖 80% 中学词汇）

| 词根              | 含义    | 中学高频示例词                                     | 单词释义      |
| --------------- | ----- | ------------------------------------------- | --------- |
| **act**         | 做、行动  | act（行动）、active（积极的）、reaction（反应）            | 行动；积极的；反应 |
| **agr**         | 田地、农业 | agriculture（农业）、agricultural（农业的）           | 农业；农业的    |
| **aud**         | 听     | audio（音频）、audience（观众）、listen（听，含 aud 词根）   | 音频；观众；听   |
| **bio**         | 生命、生物 | biology（生物）、biology（生物）、biography（传记）       | 生物；传记     |
| **ced/ceed**    | 走、前进  | succeed（成功）、process（过程）、recede（后退）          | 成功；过程；后退  |
| **cept**        | 拿、接受  | accept（接受）、except（除了）、concept（概念）           | 接受；除了；概念  |
| **circ**        | 圆、环   | circle（圆）、circumstance（环境）、cycle（循环）        | 圆；环境；循环   |
| **dict**        | 说、断言  | dictate（听写）、dictionary（字典）、predict（预测）      | 听写；字典；预测  |
| **duc/duct**    | 引导、带领 | introduce（介绍）、produce（生产）、conduct（引导）       | 介绍；生产；引导  |
| **fac/fact**    | 做、制造  | factory（工厂）、difficult（困难的）、satisfy（满足）      | 工厂；困难的；满足 |
| **fer**         | 带、拿   | prefer（更喜欢）、transfer（转移）、offer（提供）          | 更喜欢；转移；提供 |
| **fin**         | 结束、界限 | finish（完成）、final（最终的）、confine（限制）           | 完成；最终的；限制 |
| **geo**         | 地球、土地 | geography（地理）、geography（地理）、geometry（几何）    | 地理；几何     |
| **grad**        | 步、等级  | grade（年级）、graduate（毕业）、progress（进步）         | 年级；毕业；进步  |
| **ject**        | 扔、投掷  | project（项目）、inject（注射）、reject（拒绝）           | 项目；注射；拒绝  |
| **log**         | 说、记录  | logic（逻辑）、dialogue（对话）、blog（博客）             | 逻辑；对话；博客  |
| **man**         | 手、操作  | man（人）、manage（管理）、manual（手动的）               | 人；管理；手动的  |
| **port**        | 搬运、携带 | transport（运输）、import（进口）、portable（便携的）      | 运输；进口；便携的 |
| **scrib/scrip** | 写、记录  | describe（描述）、script（剧本）、write（写，含 scrib 词根） | 描述；剧本；写   |
| **vis/vid**     | 看、看见  | visit（参观）、visible（可见的）、video（视频）            | 参观；可见的；视频 |

### 二、高频前缀（15 个，改变词义方向）

| 前缀          | 含义                          | 中学高频示例词                                                            | 单词释义       |
| ----------- | --------------------------- | ------------------------------------------------------------------ | ---------- |
| **un-**     | 不、相反（最常用）                   | happy→unhappy（不高兴）、like→unlike（不像）、usual→unusual（不寻常）              | 不高兴；不像；不寻常 |
| **in-/im-** | 不、向内（in - 接辅音，im - 接 p/b/m） | possible→impossible（不可能）、complete→incomplete（不完整）、import（进口）       | 不可能；不完整；进口 |
| **dis-**    | 不、分开                        | agree→disagree（不同意）、appear→disappear（消失）、cover→discover（发现）        | 不同意；消失；发现  |
| **re-**     | 再、重新                        | do→redo（重做）、write→rewrite（重写）、turn→return（返回）                      | 重做；重写；返回   |
| **pre-**    | 在前、预先                       | view→preview（预习）、pay→prepay（预付）、school→preschool（学前）               | 预习；预付；学前   |
| **post-**   | 在后、后来                       | war→postwar（战后）、card→postcard（明信片）、graduate→postgraduate（研究生）      | 战后；明信片；研究生 |
| **pro-**    | 向前、支持                       | gress→progress（进步）、pose→propose（提议）、tect→protect（保护）               | 进步；提议；保护   |
| **anti-**   | 反对、抗                        | virus→antivirus（杀毒软件）、war→antiwar（反战的）                             | 杀毒软件；反战的   |
| **auto-**   | 自动、自己                       | mobile→automobile（汽车）、graph→autograph（签名）                          | 汽车；签名      |
| **co-**     | 共同、一起                       | work→cooperate（合作）、author→coauthor（合著者）、exist→coexist（共存）          | 合作；合著者；共存  |
| **en-/em-** | 使…、进入（en - 接辅音，em - 接 p/b）  | large→enlarge（扩大）、power→empower（授权）、body→embody（体现）                | 扩大；授权；体现   |
| **ex-**     | 向外、以前的                      | port→export（出口）、press→express（表达）、student→ex-student（往届生）          | 出口；表达；往届生  |
| **sub-**    | 在下、次等                       | way→subway（地铁）、ject→subject（科目）、marine→submarine（潜艇）               | 地铁；科目；潜艇   |
| **super-**  | 在上、超级                       | market→supermarket（超市）、man→superman（超人）、natural→supernatural（超自然的） | 超市；超人；超自然的 |
| **trans-**  | 穿过、转移                       | port→transport（运输）、fer→transfer（转移）、form→transform（变形）             | 运输；转移；变形   |

### 三、常用后缀（20 个，改变词性 / 表身份）

#### 1. 名词后缀（10 个，表人 / 物 / 抽象概念）

| 后缀              | 含义          | 中学高频示例词                                                               | 单词释义        |
| --------------- | ----------- | --------------------------------------------------------------------- | ----------- |
| **-er**         | 人 / 工具（最常用） | teach→teacher（老师）、work→worker（工人）、knife→cutter（刀具）                    | 老师；工人；刀具    |
| **-or**         | 人（表职业）      | act→actor（演员）、visit→visitor（游客）、invent→inventor（发明家）                  | 演员；游客；发明家   |
| **-ist**        | 人（表信仰 / 职业） | science→scientist（科学家）、art→artist（艺术家）、piano→pianist（钢琴家）             | 科学家；艺术家；钢琴家 |
| **-ion/-tion**  | 抽象概念        | act→action（行动）、decide→decision（决定）、invent→invention（发明）               | 行动；决定；发明    |
| **-ment**       | 抽象概念        | develop→development（发展）、agree→agreement（同意）、achieve→achievement（成就）   | 发展；同意；成就    |
| **-ness**       | 抽象性质        | kind→kindness（善良）、happy→happiness（幸福）、sad→sadness（悲伤）                 | 善良；幸福；悲伤    |
| **-ity/-ty**    | 抽象性质        | able→ability（能力）、possible→possibility（可能性）、pure→purity（纯洁）            | 能力；可能性；纯洁   |
| **-age**        | 状态 / 数量     | use→usage（用法）、pass→passage（段落）、marry→marriage（婚姻）                     | 用法；段落；婚姻    |
| **-al**         | 抽象概念        | arrive→arrival（到达）、refuse→refusal（拒绝）、survive→survival（生存）            | 到达；拒绝；生存    |
| **-ence/-ance** | 抽象概念        | differ→difference（差异）、important→importance（重要性）、appear→appearance（外貌） | 差异；重要性；外貌   |

| -------------- | ------------ | ---------------------------------------------------------------- | --------------- |
| **-ful**       | 充满… 的（最常用）   | care→careful（小心的）、help→helpful（有帮助的）、beauty→beautiful（美丽的）       | 小心的；有帮助的；美丽的    |

| **-ous/-ious** | 充满… 的        | danger→dangerous（危险的）、curious→curious（好奇的）、fame→famous（著名的）      | 危险的；好奇的；著名的     |
| **-y**         | 多… 的 / 有点… 的 | sun→sunny（晴朗的）、cloud→cloudy（多云的）、taste→tasty（美味的）                | 晴朗的；多云的；美味的     |

### 四、中学阶段记忆技巧（直接应用于学习）

1. **词根词缀组合记忆**：比如 `vis（看）+ -ible（可…的）= visible（可见的）`；`un-（不）+ happy（开心）= unhappy（不开心）`，先拆词再整合，避免死记硬背。
2. **结合课文例句记忆**：每个示例词都来自中学教材，背诵时搭配课文原句（比如 `He is an active student.`），既能记单词，又能熟悉用法。
3. **优先掌握 “高频组合”**：比如 `un-/dis- + 形容词`（表反义）、`-er/-or/-ist`（表人）、`-ion/-ment`（表抽象概念），这些组合覆盖了中学 80% 的派生词。
4. **猜词应用训练**：阅读中遇到生词时，尝试拆分（比如 `unpredictable` = `un-（不）+ pre-（预先）+ dict（说）+ -able（可…的）` → “不可预测的”），提升阅读效率。

以上词根词缀是中学英语的 “核心骨架”，掌握后能快速拓展词汇量（比如通过 `port` 可关联 `transport/import/export/portable` 等词），同时为高中英语学习打下基础，建议每天记忆 5-10 个，结合课本单词反复巩固～
